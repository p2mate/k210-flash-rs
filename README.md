# k210-flash-rs
Flashing tool for Kendryte K210 SoC

```
USAGE:
    k210-flash-rs [FLAGS] [OPTIONS] --download <download> --flash <flash> --terminal

FLAGS:
    -h, --help        Prints help information
    -t, --terminal    Start terminal after loading or flashing
    -V, --version     Prints version information

OPTIONS:
    -s, --serial <PORT>          Serial port to talk to K210 target
    -d, --download <download>    Download ELF or binary image
    -f, --flash <flash>          Flash binary image
```
